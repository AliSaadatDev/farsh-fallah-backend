# Farsh Fallah Backend

Backend API for **Farsh Fallah** project built with **Django Rest Framework**.

## Tech Stack
- Python
- Django
- Django Rest Framework

## Setup (Local)
```bash
git clone https://github.com/AliSaadatDev/farsh-fallah-backend.git
cd farsh-fallah-backend
pip install -r requirements.txt
python manage.py makemigrations
py manage.py migrate
python manage.py runserver
